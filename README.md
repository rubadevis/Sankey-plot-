🌿 Microbiome Relative Abundance → Sankey Visualization (QIIME2 → R)

An end-to-end R workflow to transform QIIME2 taxonomic output into an interactive Sankey diagram showing hierarchical microbial composition:
Kingdom → Phylum → Class → Order → Family

This repository provides a fully reproducible microbiome visualization pipeline, including:

1. Relative abundance calculation
2. Replicate averaging
3. Taxonomic cleaning
4. Interactive Sankey visualization
5. Publication-ready figure export

Designed for Bachelor’s, Master’s, and early PhD researchers working with metabarcoding or microbiome datasets.
